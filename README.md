# bookStore
An Amazon inspired book store with shopping basket, checkout and payment capabilities.

https://webshop-jp.herokuapp.com/

### Technologies
- React
- Redux
- Node.js
- Express
- AWS S3
- PaylPay SDK
- Heroku
